# TI2crud
Metodo crud feito com express, mysql e node js também foi utilizado nodemon e body-parser para melhor resultado.

## Para funcionamento:
Pasta modelagem é para criar a tabela no banco de dados
Pasta crud é a aplicação em node js, basta abrir o terminal e usar o seguinte codigo: 'npm install' em seguida o codigo 'npm run dev' / 'yarn dev'
